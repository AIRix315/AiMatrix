# Novel-to-Video Plugin

小说转视频官方插件 - Matrix Studio官方工作流插件

## 功能

- 章节拆分：自动识别小说章节结构
- 场景提取：AI提取场景和角色信息
- 资源生成：生成场景图、角色图
- 分镜脚本：AI生成视频分镜
- 配音生成：TTS配音合成
- 视频导出：合成最终视频

## 使用

1. 在Matrix Studio中导入小说文本文件
2. 启动"小说转视频"工作流
3. 按步骤完成各个阶段
4. 导出最终视频

## 开发

本插件演示了Matrix插件系统的标准用法：

- Schema注册：自定义资产类型
- 任务模板：标准化AI任务
- 链式任务：工作流编排
- MCP工具：本地工具调用
